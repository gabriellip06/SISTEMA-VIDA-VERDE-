# Sistema Vida Verde

Repositório contendo scripts SQL do sistema Vida Verde.

## Estrutura

- **inserts/** → Dados iniciais do banco  
- **selects/** → Consultas SQL  
- **updates/** → Atualizações e manipulações de dados  
- **deletes/** → Remoção de registros  

Todos os arquivos estão organizados para facilitar estudos, manutenção e execução.
